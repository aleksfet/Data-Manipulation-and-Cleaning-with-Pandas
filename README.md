# Module 10 Assignment: Data Cleaning with Pandas

## 📊 Overview
This project focuses on cleaning and preparing customer data for a fictional company called **UrbanStyle**.

The goal is to take messy, real-world data and transform it into a clean and structured dataset that can be used for analysis.

---

## 🧠 What This Project Does

- Loads raw customer data with errors and inconsistencies
- Identifies data quality issues (missing values, duplicates, formatting problems)
- Cleans and standardizes the dataset
- Converts data into correct formats
- Creates new useful features
- Generates business insights from the cleaned data

---

## 📁 Files Included

- `module10-assignment.py` → Main data cleaning script

---

## 🔧 Tools & Skills Used

- Python
- Pandas
- Data cleaning techniques
- Data transformation
- Feature creation
- Basic data analysis

---

## 🧹 Data Cleaning Steps

- Filled missing values (median, forward fill, defaults)
- Converted dates and numeric fields to correct formats
- Standardized names and categories
- Reformatted phone numbers into a consistent structure
- Removed duplicate records
- Renamed columns for clarity
- Sorted and organized final dataset

---

## 📈 Features Created

- Days since last purchase
- Average purchase value
- Purchase frequency category (High, Medium, Low)
